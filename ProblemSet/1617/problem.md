# entiation
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description


输入非负整数A和n，计算A^n.


## Input


输入有多组数据，每组数据包含两个非负整数A、n(0<=A<=99.999,  0<=n<=25).


## Output


对于每组数据，输出一行，包含一个非负整数A^n.


## Sample Input
```
0.4321 20
```
## Sample Output
```
0.00000005148554641076956121994511276767154838481760200726351203835429763013462401
```
