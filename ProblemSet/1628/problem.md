# 
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
临近期末，小K要准备抢选修课了。问题来了，一共有n个同学参加选课，共有m门课可选，至少有ai人选第i门课，选所有m门课的人至少有多少？


## Input
多组数据
每组第一行：两个整数n(0<=n<2^31)，m(0<=m<=100)
每组第二行：m个整数ai(0<=ai<=n)表示至少ai人选第i门课


## Output
每组一行，输出一个整数，表示选m门课的最少人数


## Sample Input
```
10 2
8 8
```
## Sample Output
```
6
```
