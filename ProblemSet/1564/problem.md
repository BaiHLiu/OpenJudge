Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
Houge对N！产生了兴趣，因为N！增长得非常快数字会变得非常大所以他只想知道N!末尾0的个数和N!的二进制表示中最低位1的位置。



## Input
有多行，每行输入一个数字N 1<=N<=10000000


## Output
每行输出两个数字a和b，两个数字中间隔一个空格,a为求N!末尾0的个数，b为求N!的二进制表示中最低位1的位置。


## Sample Input
```
4
5

```
## Sample Output
```
0 3
1 3

```
