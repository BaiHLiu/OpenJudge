# em 1  数字
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
定义f(x) = {比x小，不可以别x整除并且不和x互质的数的个数}(x为正整数)。
当f(x) 是奇数的时候我们称x为“奇真数”。
给出两个数x,y求区间[x,y]内的“奇真数”的个数。


## Input
第一行输入一个数N代表测试数据个数（N<=20）。接下来N行每行两个正整数x , y ( 0 < x <= y < 2^31)。


## Output
﻿对于每个测试数据输出“奇真数”的个数，每行输出一个结果。


## Sample Input
```
2
1 1
1 10
```
## Sample Output
```
0
4
```
