# ion
Time Limit: 3 Sec  Memory Limit: 64 MB


## Description
给你一个等式： X^2 + s(x,m)x – n = 0. s(x,m) 代表当x以m为底的时候，每一位的和。现在，给你n和m，你应该找到最小的正整数x满足这个等式。如果不存在，输出-1。


## Input
先输入一个T代表有T组数据，之后是T行，每一行代表一组输入，每一个数据为n(1 <= n <= 1 ^ 18) ,m (2 <= m <= 16).

## Output
先输入一个T代表有T组数据，之后是T行，每一行代表一组输入，每一个数据为n(1 <= n <= 1 ^ 18) ,m (2 <= m <= 16).

## Sample Input
```
4
4 10
110 10
15 2
432 13

```
## Sample Output
```
-1
10
3
18

```
