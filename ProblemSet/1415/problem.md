# 
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
给出一组a，b得值，分别求出a,b之间偶数的和与奇数的和（包括a，b）



## Input
输入有多组数据，每组数据有两个正整数a，b，每组数据占一行。
输入以EOF结束


## Output
分别输出a,b之间偶数的和与奇数的和（包括a，b），偶数的和与奇数得和之间有一个空格
每组测试数据占一行


## Sample Input
```
3 5

```
## Sample Output
```
4 8

```
