# 
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
小K闲的蛋疼参加了网上的一个测验。在这个测验中，有n道题，每道题答对加x分，答错扣y分，每道题答对的概率为p，求小K测验的总分大于m的概率


## Input
多组数据
每组一行，包含4个整数n(0<n<=1000),m(|m|<2^31),x,y(0<x,y<=100000)和浮点数p(0<=p<=1)


## Output
每组输出一行，包含一个浮点数，表示总分大于m的概率（精确到小数点后2位）


## Sample Input
```
2 2 2 1 0.5
2 0 2 1 0.5
```
## Sample Output
```
0.25
0.75
```
