## Description
双阶乘用“m!!”表示。
当m是自然数时，表示不超过m且与m有相同奇偶性的所有正整数的乘积。如：











## Input
输入有多组样例，每行有一个正整数n。
输入以EOF结束


## Output
输出n！！的值，每组样例占一行


## Sample Input
```
3
5
```
## Sample Output
```
3
15
```

## HINT
另0!!=1!!=1
OJ系统上测试输入结束符为EOF（End Of File），其值为-1。用scanf()把文件所有内容读完后，会读到EOF，所以可以用来判断输入是否完成，测试时可以用Ctrl+Z产生EOF。本题解法参看FAQ。
