# 女神
Time Limit: 2 Sec  Memory Limit: 256 MB


## Description
一天houge的女神突然想要养鱼，但是什么还都没买；于是，女神就对houge说如果今天你能买到一个体积是v，但因为我一点也不喜欢小数，所以边长必须都是整数，的长方形鱼缸，并且这个长方体的表面积必须最小，我就以身相许。Houge眼里哪能容下这个！！！于是立即到市区去买鱼和鱼缸。到了店里，他给老板说了要求，但是老板说，只要你能给我说出最小表面积，我就能很快做出来鱼缸，要不然，就得让houge等到黄瓜菜都凉了。
Houge想得焦头烂额，到底最小面积是多大哪？！得知此事的你想必应该能够明白houge此时的心情。So，这忙，你一定得帮啊！！！


## Input
有多组样例，每组为一个整数v(1<=v<=10^9),占用一行;以文件结束符结束。


## Output
每组样例输出一个要求的整数s，占用一行。


## Sample Input
```
1
17

```
## Sample Output
```
6
70

```
