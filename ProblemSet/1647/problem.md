Time Limit: 3 Sec  Memory Limit: 128 MB


## Description
在二维坐标轴内给出四个点，这四个点能否构成一个正方形。


## Input
第一行包括一个整数：T（1≤T≤30），表明有T组测试数据。
下面各行有8个整数：x1，y1，x2，y2，x3，y3，x4，y4（数据均在-1000,1000之间），以逆时针顺序给出四个点的坐标。


## Output
对每个样例，如果是正方形，则单独在一行内输出YES，否则，输出NO。


## Sample Input
```
2
0 0 1 0 1 1 0 1
-1 0 0 -2 1 0 2 0

```
## Sample Output
```
YES
NO

```
