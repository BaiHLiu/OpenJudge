# 水题
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
给定长度为N的字符串S，要构造一个长度为N的字符串T。起初，T是一个空串，随后反复进行下列任意操作：
1.     从S的头部删除一个字符，加到T的尾部
2.     从S的尾部删除一个字符，加到T的尾部
目标是要构造字典序尽可能小的字符串T。
限制条件：
1.1<=N<=2000

2.         字符串只包含大写英文字母


## Input
第一行输入字符串长度，第二行输入字符串


## Output
输出目标字符串，注意每80个字母一换行


## Sample Input
```
6
ACDBCB

```
## Sample Output
```
ABCBCD

```
