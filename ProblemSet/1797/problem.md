Time Limit: 3 Sec  Memory Limit: 128 MB


## Description

	一只蜗牛从井底向上爬，爬一秒休息一秒，爬行速度是u米/秒，休息时会下滑，下滑速度为d米/秒，井的高度是n米，问蜗牛爬到顶部要多久。


	


	


## Input
输入三个正整数，n,u,d,含义如上所述，输入保证在int范围之内。输入以三个0结束。

## Output
输出一个数字，表示蜗牛多久能爬上来。如果蜗牛永远也不可能爬上去，输出NEVER!。

## Sample Input
```
10 10 1
10 5 6
10 5 4
0 0 0
```
## Sample Output
```
1
NEVER!
11
```
