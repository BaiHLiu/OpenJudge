## Description
      大家都知道A+B是最简单的题，现在同样有道A+B的题目请你解决。



## Input
      第一行输入一个整数T（T<=20），表示接下来有T组测试用例。

      在每组测试用例中，每行输入两个正整数a、b(a、b可能很大)。它们的进制未知，可能位于2到10之间。


## Output
      对于每组测试用例，输出测试用例编号和a+b在10进制下的最小值。保证结果小于2^64。


## Sample Input
```
2
0 0
1 1

```
## Sample Output
```
Case 1: 0
Case 2: 2

```
