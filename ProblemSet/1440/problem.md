# em 6  多项式求和
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
多项式描述如下：
1 - 1/2 + 1/3 - 1/4 + 1/5 - 1/6 ……
先请你求出多项式前n项的和。


## Input
﻿第一行输入一个数T代表测试数据个数（T<=1000）。接下来T行每行1个数代表n(0<=n< 2^31)。


## Output
对于每个输入样例，输出多项式和的结果(结果精确到小数点后两位)。每行输出一个结果。


## Sample Input
```
2
1
2

```
## Sample Output
```
1.00
0.50

```
