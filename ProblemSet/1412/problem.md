Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
我们假设有一个整数K,需要你寻找两个自然数M和N（M>N），使得K^M和K^N(1000 <= K^N)的末尾三位数相等，如果存在这样的M和N,则输出一组N+M的值,使得N+M尽可能小,否则输出"No Answer."



## Input
输入有多组测试数据,每行为一个K值.
输入以EOF结束


## Output
若存在合法的N和M,输出M+N的最小值,否则输出"No Answer.".


## Sample Input
```
2
```
## Sample Output
```
120
```
