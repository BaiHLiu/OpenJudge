# 
Time Limit: 3 Sec  Memory Limit: 128 MB


## Description
用C语言画些任意大小的笑脸来。画法见Sample。


## Input
T组数据，每组数据只有一个数N（1≤T≤20,1≤N≤20），表示笑脸的边长大小。


## Output
把笑脸画出来吧。
注意：不要任何多余的空格和空行，每两组数据间输出一个空行。


## Sample Input
```
3
1
2
3

```
## Sample Output
```
/\ /\
  _

 /\    /\
/  \  /  \

    __

  /\       /\
 /  \     /  \
/    \   /    \


      ___
```
