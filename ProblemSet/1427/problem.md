# hi爱开方
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
定义可开方数：一个整数执行开方运算后依旧是一个整数如1，4，9，.....
Dearchi想统计x到y之间有多少个可开方数（包括x和y），你帮帮他吧~


## Input
输入x和y （0<x<y<1000)
当输入0 0时结束


## Output
输出统计结果  


## Sample Input
```
1 10
1 100
0 0
```
## Sample Output
```
3
10

```
