# 多少个最大值？
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
输入若干个int类型的整数，求它们的最大值及其个数。

## Input
输入 若干个int类型的整数，至文件尾为止。

## Output
输出只有一行：There are # maximum number &.
其中#是最大值的个数，&是最大值。

## Sample Input
```
1 2 3 4 5 6 6 5 4 3 2 1 6 6 6

```
## Sample Output
```
There are 5 maximum number 6.

```

## HINT
不能使用数组。
